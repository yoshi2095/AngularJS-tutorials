Anchor scroll and location of an element



In this tutorial, we are making a scroll from one position of the page to another using the "$anchorscroll" object of AngularJS. We are giving an id to the individual elements (to which we want to be scrolled to) and attaching them to a click event. This click calls a function in our controller, and passes the name of the id of the element we want to scroll to, in the form of a string. the function then takes the argument and passes it into the hash function of the $location provider. The hash function takes the value in the parameter and appends it in the url with a #. Then the $anchorScroll() function is called which will read the appended hashed value and scroll to the element of the page which has that as its id.



app.js:



	
		.controller("anchorScrollAndLocationController", function($scope, $location, $anchorScroll){
			$scope.scrollTo = function(scrollLocation){
							$location.hash(scrollLocation); //hash function will take the value in scrollLocation and
															// will append it to the url with a hash (#)
							$anchorScroll.yOffset = 20; //this is used to set top padding (from the browser window)
														//to the anchorScrolled element.
							$anchorScroll(); // this will read the appended hashed value and scroll to the element
											 // of the page which has that as its id.
						}
		});


html:



	
		<md-button id="top" class="md-accent md-raised" ng-click="scrollTo('bottom')">Go to the bottom of the page</md-button>

		<div>
		    <!-- text content -->
		</div>

		<md-button id="bottom" class="md-accent md-raised" ng-click="scrollTo('top')">Go to the top of the page</md-button>
